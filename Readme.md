# VPS Rsync Sync GUI

GUI tool untuk sinkronisasi folder lokal ke VPS menggunakan rsync + SSH.

## Fitur
- Multi VPS profile
- SSH key manager
- Rsync cepat & resumable
- Windows & macOS

## Cara Pakai
1. Generate SSH key
2. Upload public key ke VPS
3. Buat profile VPS
4. Pilih folder lokal
5. Start Sync

## Windows
Install Git for Windows (rsync included)

## macOS
rsync sudah tersedia
